# Numerical-and-Statistical-methods

A collection of numerical and statistical methods implemented in Python.

## Methods Included

| File | Method |
|---|---|
| `bisection.py` | Bisection Method |
| `regula_falsi.py` | Regula Falsi (False Position) Method |
| `newton_raphson.py` | Newton-Raphson Method |
| `gauss_elimination.py` | Gaussian Elimination |
| `gauss_seidel.py` | Gauss-Seidel Iterative Method |
| `jacobi.py` | Jacobi Iterative Method |
| `newton_forward.py` | Newton's Forward Interpolation |
| `newton_backward.py` | Newton's Backward Interpolation |

Each program takes input from the user at runtime.

## Topics Covered

- Root Finding Methods
- System of Linear Equations
- Interpolation Techniques
